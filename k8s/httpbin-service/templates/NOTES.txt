HTTPBin is a simple HTTP request/response testing service.

To access the application:
{{- if eq .Values.service.type "ClusterIP" }}
  kubectl port-forward svc/{{ include "httpbin-service.fullname" . }} {{ .Values.service.port }}:{{ .Values.service.port }}
  Then open http://localhost:{{ .Values.service.port }}
{{- else if eq .Values.service.type "NodePort" }}
  export NODE_PORT=$(kubectl get --namespace {{ .Release.Namespace }} -o jsonpath="{.spec.ports[0].nodePort}" services {{ include "httpbin-service.fullname" . }})
  export NODE_IP=$(kubectl get nodes --namespace {{ .Release.Namespace }} -o jsonpath="{.items[0].status.addresses[0].address}")
  echo http://$NODE_IP:$NODE_PORT
{{- else if eq .Values.service.type "LoadBalancer" }}
  kubectl get svc {{ include "httpbin-service.fullname" . }} -w
{{- end }}

Useful endpoints:
  /get          - Returns GET request data
  /post         - Returns POST request data
  /status/:code - Returns given HTTP status code
  /headers      - Returns request headers
  /ip           - Returns origin IP
  /delay/:n     - Delays response by n seconds
